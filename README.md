## react-components
자주 사용하는 리액트 컴포넌트를 모아놓은 프로젝트.

---
## Dependencies
- node v12.13.1
- react
- typescript
- babel
- webpack
- storybook

```bash
  $ cd {projectPath}
  $ npm install
```

---
## Execute
```bash
  $ cd {projectPath}
  $ npm run storybook
```
